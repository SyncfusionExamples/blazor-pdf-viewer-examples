# Annotations
This section contains samples for publishing the Blazor Webapps sample into the Azure docker container

Documentation: https://help.syncfusion.com/document-processing/pdf/pdf-viewer/blazor/deployment/azure-container-deployment

## Project pre-requisites
To run this application, you need Visual Studio 2019 or later versions.
An Azure subscription and permission to create resource groups, ACR instances, and App Services.

Note: Make sure to install the Wasm-tools and the workloads to make sure the pdf viewer component render properly in the Webapps Wasm sample.
